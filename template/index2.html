To calculate V1 to V28 for credit card fraud detection using PCA:
1. Standardize Data: Transform features to have a mean of 0 and a standard deviation of 1.
2. Compute Covariance Matrix: Assess how features vary together.
3. Find Eigenvalues/Eigenvectors: Solve \(\Sigma v = \lambda v\) to identify key directions of variance.
4. Sort Eigenvalues: Rank them and select the top ones.
5. Transform Data: Create new features using the top eigenvectors: \(Y = ZW\).
